1.代码简介
	代码在code文件夹中，官方数据存放在data文件夹中，生成的feature文件在feature文件夹中，生成的线上提交结果存在submit文件夹中。

	每种特征生成后存成一个单独的文件，在训练模型之前一次融合所有特征。

2.参数、函数介绍
	genFea：生成特征时为1，训练特征时为0.
	fea（）：此函数用于选择性生成特征文件。已生成的特征请注释以节省时间。此函数分别调用不同的特征生成函数，fea1（）~fea7（）
	conbine_features（）：此函数用于训练模型前一次性融合所有特征，供模型训练使用。
	cv：做本地cv，看特征以及模型参数效果时为1；生成线上结果时为0.
	num_round：做本地cv时设为5000.做完本地cv后，假设在第n轮获取到最大test-auc。在生成线上结果时num_round设为n。
